iSpec

Sistema de Inspeção de Equipamentos de Incêndio

Versão 1.0.0

O iSpec é um sistema web para gerenciamento de inspeções de equipamentos de combate a incêndio, desenvolvido como Trabalho de Conclusão de Curso (TCC). O sistema permite o cadastro de clientes, equipamentos, realização de inspeções com checklist específico por tipo de equipamento, agendamento de visitas e geração de relatórios em PDF e Excel.
